إضافة بادجات نوع العقار في صفحة إضافة الطلب (/dashboard/customers-hub/requests/add): (1) "عقار موجود بالفعل" — عند الاختيار يظهر dropdown لاختيار عقار أو أكثر من قائمة العقارات (GET /properties)، مع عرض المختارين كـ badges قابلة للإزالة.
